# flypy_byrd
flappy bird clone in pygame

![demo](./demo.gif)

### To run in your device you may need to change this values

```
if __name__=="__main__":
    width = 355
    height = 520
    conf.change_res(width,height)
    conf.p_col_step=5
    conf.n_pillar=3
    conf.hr_space=250
    conf.pillar_w = 50
    conf.pillar_cap = 25
    conf.vr_space = 100//2
    conf.bird_size = 25
    conf.speed=200
    conf.energy_rate=4
    conf.bird_speed=300
    conf.bg_pos = 200
    conf.cloud_r = (50,100)
    conf.cloud_sp=10
    conf.cheat=not True
    conf.tree_r=(25,50)
    conf.tree_sp = 2
    conf.bg_h_sub = 20
    conf.vr_variance=10
    run()
```
